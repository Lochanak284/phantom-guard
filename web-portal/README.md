# Phantom Guard - Web Portal

This is Module 4 of Phantom Guard - React web dashboard.

## Backend URL
https://phantom-guard-backend.onrender.com

## API Endpoints
- POST /auth/register
- POST /auth/login
- POST /device/register
- GET /device/status?device_id=
- POST /agent/activate
- POST /agent/catch
- POST /evidence/upload
- GET /evidence/get?device_id=

## Setup
- npm install
- npm start
